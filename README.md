CASE STUDY 6: Secure Password Handling (DevSecOps Intro)
 Objective
Understand secure coding practices.
 Scenario
Students mistakenly push password to GitHub.
 Steps
1.	Create config file with password
2.	Commit & push
3.	Teacher explains risk
4.	Use .gitignore
5.	Remove secret and recommit
 Mindful Activity
Think before committing sensitive data.
 Learning Outcome
•	Security awareness
